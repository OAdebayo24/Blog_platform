# Blogging Platform

A blogging platform where users can write, draft, publish, and explore articles — with real-time data syncing, a responsive UI, and interactive analytics charts.

## Features

- **Real-time Database**:  Posts data synced live with Firebase Firestore
- **Interactive analytics**:  Charts created by Highcharts showing post published, and created by users in the last 7 days
- **Responsive design**: UI built with Tailwind CSS, works seamlessly on all screen sizes

## Tech Stack

- **frontend**: React
- **Backend**: firebase 
- **Data visualization**: High chart
- **Styling**: TailwindCSS

## Project structure
```
src/
├── components/       # Reusable UI components
├── pages/            # Route-level page 
├── firebase/         # Firebase config and 
├── charts/           
└── styles/          
```
